# 文件
## 列出文件，按照时间排序
**列出文件，按修改时间排序（最新的在最上面）**:
```
ls -lt
```
**列出文件，按修改时间排序（最旧的在最上面）**:
```
ls -ltr
```
如果您的目标是查找特定类型的文件（例如XML格式的审计文件），您可以使用`grep`命令进一步筛选结果，例如：
```
ls -lt | grep '.xml'
```
